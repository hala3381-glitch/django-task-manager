# Django Task Manager (To-Do List)

A production-ready Task Management web application built with Django. This platform helps users efficiently manage their daily tasks, offering an intuitive lifecycle tracking system with dynamic task statuses.

## 🚀 Features
* *Task Lifecycle Management*: Track tasks seamlessly through three distinct states: Not Started, Working on It, and Completed.
* *State Management*: Built with a clean Backend logic using Django Model choices to handle dynamic status updates correctly.
* *Efficient Database Schema*: Uses a structured relational database layout optimized for quick CRUD (Create, Read, Update, Delete) operations.
* *User-Centric Design*: Simple and fast workflow for adding, modifying, and tracking productivity goals.

## 🛠️ Tech Stack
* *Backend*: Python, Django Framework
* *Database*: SQLite (Development)
* *Tools*: Git, GitHub
